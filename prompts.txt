Project Setup and Prompts Used Throughout the Project:

1. Setup:
   pip install uv
   mkdir ekantipur_scraper && cd ekantipur_scraper
   uv init
   uv add playwright
   uv run playwright install chromium

2. Goal:
   Create a Python 3.11+ Playwright script using uv that scrapes https://ekantipur.com and saves valid JSON to output.json.

   Required tasks:
   1. Go to the Entertainment section (मनो रञ्जन)
   2. Extract the top 5 news articles with:
      - title
      - image_url
      - category
      - author (null if missing)
   3. Extract the "Cartoon of the Day" / "व्यंग्यचित्र" section with:
      - title
      - image_url
      - author

   Required output structure:
   {
     "entertainment_news": [
       {
         "title": "...",
         "image_url": "...",
         "category": "...",
         "author": "..."
       }
     ],
     "cartoon_of_the_day": {
       "title": "...",
       "image_url": "...",
       "author": "..."
     }
   }

   Technical requirements:
   - Use Python sync Playwright
   - Use robust selectors and stable attributes
   - Avoid fragile XPaths unless necessary
   - Handle missing elements safely
   - Wait properly for dynamic content
   - Write clean, readable code with meaningful comments
   - Use ensure_ascii=False when writing JSON
   - Keep code modular with helper functions

   Working approach:
   - Do NOT generate the entire solution at once
   - First generate a clean scraper.py skeleton with:
     - main()
     - browser setup
     - navigation placeholder
     - helper functions
     - JSON writing
   - Then manually inspect selectors in DevTools
   - Then iteratively complete extraction logic

   Expected project files:
   - scraper.py
   - output.json
   - pyproject.toml
